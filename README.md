## Getting Started

What to do to start the project

### Prerequisites

Update node package manager with this command:
```sh
  npm install npm@latest -g
```
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/CodecoolGlobal/react-for-a-cause-react-ignacyD
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the project
   ```sh
   npm start
   ```
